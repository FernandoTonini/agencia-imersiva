Passo 3 de 5

Adicionar o "robô" da planilha

Agora vamos adicionar o código que recebe os leads e cria as linhas automaticamente. Não se assuste com o código — você só precisa copiar e colar.

  1. Na planilha, vai em Extensões → Apps Script (no menu superior)
  2. Vai abrir uma nova aba com um editor de código
  3. Apaga o que tiver lá (geralmente function myFunction() {})
  4. Cola o código abaixo no lugar
function doPost(e) { const sheet = SpreadsheetApp.getActiveSheet(); const data = JSON.parse(e.postData.contents); const a = data.answers || {}; const getVal = (k) => { const v = a[k]; if (!v) return ''; if (v.value) return v.value; if (v.labels) return v.labels.join(' | '); return v.label || ''; }; sheet.appendRow([ new Date(data.createdAt), data.id, (data.tier || '').toUpperCase(), data.score, data.contact?.name || '', data.contact?.biz || '', data.contact?.phone || '', data.contact?.email || '', data.contact?.ig || '', getVal('stage'), getVal('bottleneck_early') || getVal('bottleneck_grow') || getVal('bottleneck_scale') || getVal('why_now'), getVal('mkt_history'), getVal('budget'), getVal('urgency'), getVal('decision'), getVal('priorities'), getVal('context') ]); return ContentService.createTextOutput(JSON.stringify({ok: true})) .setMimeType(ContentService.MimeType.JSON); }
Depois de colar: clica no ícone de 💾 disquete (Salvar) no topo do editor.
Se pedir nome do projeto, pode chamar de Webhook Leads.
Passo 4 de 5

Publicar como Web App

Última coisa que falta dentro do Apps Script: publicar pra gerar a URL que o quiz vai chamar.

Ainda no editor do Apps Script:
  1. Clica no botão azul "Implantar" (canto superior direito)
  2. Escolhe "Nova implantação"
  3. Clica no ⚙️ (engrenagem) e escolhe "Aplicativo da Web"
  4. Em "Descrição": pode deixar em branco ou escrever Webhook leads
  5. Em "Executar como": deixa "Eu mesmo"
  6. Em "Quem tem acesso": muda para "Qualquer pessoa" ⚠️ (esse é o pulo do gato — sem isso não funciona)
  7. Clica em "Implantar"
  8. Vai pedir autorização. Clica "Autorizar acesso"
  9. Escolhe sua conta. Se aparecer aviso "Não verificado": clica em "Avançado" → "Acessar (não verificado)" → "Permitir"
  10. Vai aparecer uma tela com "URL do aplicativo da Web". É essa URL que vamos usar no próximo passo.
⚠️ Importante: a URL começa com https://script.google.com/macros/s/... e termina com /exec. Copia ela INTEIRA.
